# ESP32 天气时钟接线说明

日期：2026-07-12

## 当前硬件判断

- 主控：照片中为 30P ESP32 DevKit 类开发板，板上丝印可见 `3V3`、`GND`、`D18`、`D23`、`D5`、`D2`、`D4` 等常用引脚。
- 屏幕：用户确认是 1.54 寸 240x240 SPI TFT。第一版固件按 ST7789 240x240 配置。
- 供电：第一版只用 USB 给 ESP32 供电，屏幕从 ESP32 的 `3V3` 取电。

## 推荐接线

请先确认屏幕背面 8 个针脚丝印。如果丝印类似 `GND VCC SCL SDA RES DC CS BLK`，按下表接线：

| 屏幕针脚 | ESP32 针脚 | 说明 |
| --- | --- | --- |
| `GND` | `GND` | 共地 |
| `VCC` | `3V3` | 优先接 3.3V，不要先接 5V |
| `SCL` / `SCK` / `CLK` | `D18` / `GPIO18` | SPI 时钟，ESP32 VSPI SCK |
| `SDA` / `DIN` / `MOSI` | `D23` / `GPIO23` | SPI 数据，ESP32 VSPI MOSI |
| `RES` / `RST` | `D4` / `GPIO4` | 屏幕复位 |
| `DC` / `A0` | `D2` / `GPIO2` | 数据/命令选择 |
| `CS` | `D5` / `GPIO5` | SPI 片选 |
| `BLK` / `LED` | `3V3` | 背光常亮；如果暂时不接，程序仍会运行，但屏幕可能完全黑屏 |

如果屏幕针脚带 `MISO`，第一版可以不接。天气时钟只写屏幕，不需要从屏幕读数据。

## 上电前检查

1. 确认 `VCC` 接 `3V3`，不是 `VIN` 或 `5V`。
2. 确认 ESP32 和屏幕 `GND` 已连接。
3. 确认没有把 `SCL/SDA` 当作 I2C 接到 `D22/D21`。这块屏幕按 SPI 接，不按 I2C 接。
4. 如果屏幕背面丝印不是上述名称，先拍清楚背面针脚文字再接。

## 后续固件配置目标

第一轮屏幕点亮测试优先使用以下配置：

```cpp
#define TFT_MOSI 23
#define TFT_SCLK 18
#define TFT_CS    5
#define TFT_DC    2
#define TFT_RST   4
```

驱动芯片先按 `ST7789` 测试；如果显示花屏、颜色错乱或偏移，再调整颜色顺序、偏移或切换其它 240x240 驱动配置。
